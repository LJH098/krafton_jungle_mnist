# 개선 1. 최적 하이퍼파라미터 탐색

## 실험 목적

기본 모델은 안정적으로 98%대 정확도에 도달하였으나, 추가적인 성능 향상을 위해서는 단일 설정에 의존하기보다 학습률, Dropout 비율, batch size, 은닉층 구조를 함께 조정할 필요.

이에 따라 validation 데이터를 기준으로 여러 하이퍼파라미터 조합을 탐색하고, 상위 5개 조합을 20 epoch 동안 재학습한 뒤 test 데이터에서 최종 성능 비교.

---

## 탐색한 하이퍼파라미터

| 항목 | 후보 |
| --- | --- |
| 학습률 | 0.003, 0.001, 0.0007, 0.0005, 0.0003 |
| Dropout 비율 | 0.2, 0.3, 0.4, 0.5 |
| Batch size | 64, 128, 256 |
| 은닉층 구조 | 1층~4층, 다양한 뉴런 수 조합 |
| 기준 지표 | Validation Accuracy |

---

## 실험 절차

1. 여러 하이퍼파라미터 조합을 validation accuracy 기준으로 탐색
2. 상위 5개 조합 선정
3. 각 조합을 train 데이터로 20 epoch 재학습
4. 최종 test 데이터로 정확도 비교

## 결과 해석

가장 높은 test accuracy는 Top 1 조합에서 도출.

```
은닉층 구조: 1024 -> 512 -> 256
학습률: 0.0007
Dropout 비율: 0.3
Batch size: 64
Test Accuracy: 98.53%
```

Top 2 조합은 final train loss가 더 낮았으나, test accuracy는 Top 1 조합이 더 높게 측정. 이는 train loss가 낮다고 해서 test accuracy가 항상 가장 높아지는 것은 아님을 의미.

즉, Top 2는 학습 데이터에 대해 더 낮은 손실을 보였으나, test 데이터 기준의 일반화 성능은 Top 1 조합이 더 우수.

---

## 결론

하이퍼파라미터 탐색 결과, 단순히 모델 크기를 키우는 것보다 은닉층 깊이, Dropout 비율, 학습률, batch size의 조합이 성능에 중요한 영향.

특히 3개의 은닉층을 사용한 `1024 -> 512 -> 256` 구조가 가장 높은 test accuracy를 달성.

```
최종 선택 모델: 1024 -> 512 -> 256
Test Accuracy: 98.53%
```

다만 정확도 향상 폭은 제한적. 이는 MNIST에서 완전연결 신경망만으로는 98%대 후반 이후의 성능 향상이 점차 어려워지며, 99% 이상을 목표로 할 경우 CNN과 같은 이미지 구조에 더 적합한 모델 도입이 필요함을 시사.
